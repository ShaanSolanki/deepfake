# User Guide - Deepfake Detection & Response Platform

## Quick Start

1. Open http://localhost:5000 in your web browser
2. Upload a video file
3. Click "Analyze Video"
4. Follow the post-detection guidance

## Navigation

### Detection Tab

Main interface for uploading and analyzing videos

### Awareness Tab

Educational content about deepfakes, ethics, and best practices

### Mode Toggle

Switch between Individual Mode and Organization Mode

## Features Guide

### 1. Video Upload

- **Drag & Drop**: Drag video file onto upload area
- **Click to Browse**: Click upload area to select file
- **Supported Formats**: MP4, AVI, MOV
- **Max Size**: 100MB

### 2. Detection Results

#### REAL Result (Green)

- Video appears authentic
- High confidence in authenticity
- Still verify source and context

#### FAKE Result (Red)

- Video appears to be deepfake
- Follow immediate action steps
- Do NOT share the video

#### UNSURE Result (Yellow)

- Analysis inconclusive
- May need expert review
- Consider video quality issues

### 3. Risk Assessment

Answer 4 simple questions:

1. Does video impersonate you/someone you know?
2. Potential for financial fraud?
3. Has video been publicly shared?
4. Could it damage reputation?

**Risk Levels:**

- **Low (1 checked)**: Platform reporting
- **Medium (2-3 checked)**: Legal + platform reporting
- **High (4 checked)**: Immediate legal action

## Troubleshooting

### "Unable to Determine" Result

- Video quality too low
- No clear faces detected
- Technical limitations
- Try different video or seek expert analysis

### Upload Fails

- Check file size (max 100MB)
- Verify file format (MP4, AVI, MOV)
- Ensure stable internet connection
- Try different browser

### Slow Analysis

- Large files take longer
- Server processing multiple requests
- Normal for high-quality videos
- Wait for completion
